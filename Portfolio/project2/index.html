Restaurant Website

Project Description

This is a website designed for "Kunle Delights", an authentic Nigerian cuisine restaurant. The site includes:

  • Signature dishes straight from the motherland showcased with vibrant food photography
  • Menu's with detailed descriptions
  • Online ordering capabilities
  • Responsive design that works on mobile and desktop devices

This project demonstrates my ability to create attractive food service websites with clean layouts and easy navigation.
