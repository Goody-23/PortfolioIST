Record Store Website

Project Description

This is a website designed for a vinyl record store called "Goody Vinyls". The site features:

  • Featured albums section showcasing the newest releases
  • Current inventory listing with sorting options
  • Clean, modern interface for browsing vinyl records
  • Easy navigation for customers to find their favorite music

This project was built using HTML, CSS, and basic JavaScript to create an interactive shopping experience.
